A laptop-free dev setup

The phone is the computer. The glasses are the screen. A folding keyboard is the input. On a train, a plane tray table, or a café — a full Linux desktop with VS Code, anywhere.

📱

Phone is the computer

Android's Linux Terminal runs a full Debian VM (AVF/KVM). On Pixel it includes GPU-accelerated GUI apps — a real VS Code window.

🕶️

Glasses are the screen

USB-C DisplayPort drives the AR glasses as a 1200p wearable monitor — Viture Beast / Luma Ultra or XREAL 1S.

⌨️

Folding keyboard

A tri-fold Bluetooth keyboard with trackpad pairs in Android and is forwarded into the VM. One device replaces keyboard + mouse.

🐧

Linux + VS Code

labwc (Wayland) + VS Code natively, or code-server in the browser as a rock-solid fallback. Tile windows for multi-pane.

Multi-screen, the honest way

A phone outputs one display. The glasses anchor and enlarge it; your multiple panes come from tiling in the Wayland compositor — not from any glasses app.

XREAL's X1 chip anchors in hardware (no app); Viture anchors via SpaceWalker.
