pocketdesk

A pocket-sized Linux dev workstation — your phone is the computer, AR glasses are the screen, a folding keyboard is the input. A real desktop, in your pocket. No laptop.

Android Linux Terminal (AVF/KVM) Debian VM · labwc · VS Code (Wayland) USB-C DisplayPort → AR/XR glasses folding Bluetooth keyboard + trackpad model-agnostic (Pixel today) community knowledge base
Documentation GitHub

pocketdesk documents a fully mobile development setup: write code in VS Code on Linux (Debian / Wayland) running directly on a Pixel phone, displayed on AR glasses, driven by a folding Bluetooth keyboard with trackpad. Train, plane tray table, or café — a real desktop in your pocket, with no laptop to carry.

It works because three recent pieces line up: Android's Linux Terminal (Android 16+) runs a full Debian VM through the Android Virtualization Framework — on Pixel with GPU-accelerated GUI apps, so a real VS Code window, not just a shell; USB-C DisplayPort Alt Mode lets the phone drive the glasses as a wearable 1200p monitor; and Bluetooth HID keyboard/mouse pair at the Android level and forward into the VM. The chain depends on Android + AVF, not a specific phone — the guides work on a Pixel today and transpose unchanged as new models ship.
